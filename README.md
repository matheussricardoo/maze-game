### 📦 Dependencies | Dependências

```bash
python: >= 3.7
pygame: >= 2.0.0
```

### 🚀 Getting Started | Começando

```bash
# Clone the repository | Clone o repositório
git clone https://github.com/matheussricardoo/maze-game.git

# Navigate to project directory | Navegue até o diretório do projeto
cd maze-game

# Install dependencies | Instale as dependências
pip install pygame

# Run the game | Execute o jogo
python labirinto_sonoro.py
```

### 🎮 Controls | Controles

<div align="center">

| Key | Action | Ação |
|:---:|:-------|:-----|
| ↑/↓/←/→ | Move player | Mover jogador |
| ESC | Back to menu | Voltar ao menu |
| R | Restart after death | Reiniciar após morte |
| ENTER | Next level (after win) | Próximo nível (após vitória) |

</div>
